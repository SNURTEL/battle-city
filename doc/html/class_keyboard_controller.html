Detailed Description

Translates SFML's window events (or, more precisely, KeyPressed, KeyReleased and NullEvent events) to game's internal events. Derived from SimpleSubscriber -> can (and should) be connected to Clock to trigger key event pooling

void KeyboardController::fetchKeyboardEvent ( ) const

Pools an event from SFML's window, translates it to game's internal Event equivalent, and appends it to global event queue Can fetch either:

  • KeyPressed (from sf::Event::KeyPressed)
  • KeyReleased (from sf::Event::KeyReleased)
  • NullEvent (if none of the above was registered)
