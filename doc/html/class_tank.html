Detailed Description

Base class for tank representation.

Tanks are derived from Entity and have some additional tank-specific fields Additionally, tanks inherit from SimpleSubscriber and can subscribe to Bullet instances fired by them (to disable firing when a bullet was already fired)

Tanks provide no input validation whatsoever and should only be manipulated by EntityController.
