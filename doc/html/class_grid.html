Detailed Description

Represents a grid of tiles.

Each tile is represented by a TileType enum value, and it's attributes can be accessed using the TileManager object This is done in order to avoid creating up to 52^2 objects with nearly identical attributes 'Empty' fields on the map should be filled with NullTiles
