Detailed Description

Class responsible for bullets updates and render.

Fetchs bullets positions and redners its on the screen Defines enum TextureType for matching textures with bullet type Defines struct RenderObject for storing matched texture together with bullet coords Stores paths to textures in a std::map<TextureType, std::string>
