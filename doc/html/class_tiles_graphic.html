Detailed Description

Class responsible for board updates and render.

Fetchs tiles positions and redners its on the screen Defines enum TextureType for matching textures with tile type Defines struct RenderObject for storing matched texture together with tile coords Stores paths to textures in a std::map<TextureType, std::string>
