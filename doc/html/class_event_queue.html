Detailed Description

template<class E>
class EventQueue< E >

Singleton class wrapper for std::queue<E>.

Stores and queues events wrapped in std::unique_ptr. Singleton instance can be accessed with ::instance(). Different instances are created for every each type (E). Cannot be inherited from (::instance() would always return the base class)

NOTE: When testing event-generating objects, call EventQueue::instance()->clear() or assert EventQueue::instance()->isEmpty() to guarantee starting and ending the test with an empty queue
